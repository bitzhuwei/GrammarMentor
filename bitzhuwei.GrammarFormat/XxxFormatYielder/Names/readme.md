## Html

|----------|----------|--------------------|----------|

| \>       | 大于号   | \&gt;              | \&\#62;  |
| &        | 和号     | \&amp;             | \&\#38;  |
| "        | 引号     | \&quot;            | \&\#34;  |
| '        | 撇号     | \&apos; (IE不支持) | \&\#39;  |

| £        | 镑       | \&pound;           | \&\#163; |
| ¥        | 日圆     | \&yen;             | \&\#165; |
| §        | 节       | \&sect;            | \&\#167; |
| ©        | 版权     | \&copy;            | \&\#169; |
| ®        | 注册商标 | \&reg;             | \&\#174; |
| ×        | 乘号     | \&times;           | \&\#215; |
| ÷        | 除号     | \&divide;          | \&\#247; |
